### 목적

콜로소 강의를 들으려고 할 때 크롬이 열려있다면 다음과 같은 메시지가 뜬다

> "녹화 프로그램(크롬 확장프로그램)이 감지되었습니다. 재생을 종료합니다.

그래서 사파리로 들어야하는데, 기존에 크롬을 모두 닫아줘야한다.
물론, 크롬 '방문기록'탭을 통해 기록이 남아있지만, 가끔 닫았던 탭을 다시 찾지 못하는 경우도 발생한다.

<br>

이 크롬익스텐션은 다음과 같이 사용할 수 있다.
